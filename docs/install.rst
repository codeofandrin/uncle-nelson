Installation
==============

This page guides you through the installation of the uncle-nelson library.

Prerequisites
--------------
uncle-nelson requires **Python 3.9 or higher**. Any version below 3.9 is not supported.

Installing
-------------

To install the latest stable version, use ::

    pip install -U uncle-nelson

To install the development version (may be unstable), use ::

    pip install -U git+https://github.com/codeofandrin/uncle-nelson
